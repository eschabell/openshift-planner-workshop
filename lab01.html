Lab 1

Install the

Business Resource Planner

Lab Goal

To install Business Resource Planner component.

Workshop prerequisites

  • The following JBoss products:
    • JBoss BRMS Business Resource Planner 6.1 installer
    • JBoss EAP 6.4 installer
    • JBoss Developer Studio 8.1.0
  • Java 1.7 or 1.8
  • Maven 3.2 or higher
  • Basic understanding of Java development.
  • Basic understanding of Drools Rule Language (DRL).

Download product

JBoss BRMS Business Resource Planner

Installation

Business Resource Planner

  • Download and unzip.
  • Add products, see installs/README for details
    • $ cp jboss-brms-6.1.0.GA-planner-engine.zip business-resource-planner-workshop/installs/
    • $ cp jboss-brms-6.1.0.GA-planner-engine.zip business-resource-planner-workshop/installs/
  • Run init.sh or init.bat
    • $ ./init.sh
  • Read and follow instructions shown during installation.
